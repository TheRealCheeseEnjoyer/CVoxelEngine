# A Voxel Engine made in pure C

## Dependencies
+ [cglm](https://github.com/recp/cglm)
+ [glad](https://glad.dav1d.de/#language=c&specification=gl&api=gl%3D3.3&api=gles1%3Dnone&api=gles2%3Dnone&api=glsc2%3Dnone&profile=core&loader=on)
+ [glfw](https://www.glfw.org/)
+ [freetype](https://freetype.org/)

## Controls
+ WASD: move
+ E: open inventory
+ Left Click: destroy block
+ Right Click: place block
+ Mouse Wheel: select hotbar slot
